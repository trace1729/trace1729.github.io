下载clash

  • 搜索clash for windows, 选择 Clash.for.Windows.Setup.0.19.25.exe 下载，安装

配置clash （windows）

  • 从机场官网将地址一键导入clash，打开以下五个选项

    ​

  • 进入 profile, 选择 在文件夹打开配置文件

    ​

  • 将文件夹中的配置文件，复制到linux 主机上的特定目录下（假定我们放置在了/home/trace/clash 文件夹中）

linux 端 clash 配置

下载clash

  • 进入clash的github relesse页面, 下载clash-freebsd-amd64-v3-v1.11.4.gz，并上传到linux主机的clash文件夹中

解压文件

  • gzip -d {filename} => .gz 后缀
  • tar -xf {filename} => .tar.gz 后缀

添加可执行权限

  • chmod +x clash-freebsd-amd64-v3-v1.11.4
